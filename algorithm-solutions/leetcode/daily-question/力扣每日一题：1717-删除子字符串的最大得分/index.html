# 📘 题目描述

给你一个字符串 s 和两个整数 x 和 y 。你可以执行下面两种操作任意次。

  • 删除子字符串 "ab" 并得到 x 分。
    • 比方说，从 "cabxbae" 删除 "ab" ，得到 "cxbae" 。
  • 删除子字符串 "ba" 并得到 y 分。
    • 比方说，从 "cabxbae" 删除 "ba" ，得到 "cabxe" 。

请返回对 s 字符串执行上面操作若干次能得到的最大得分。

  • 1 <= s.length <= 10^5
  • 1 <= x, y <= 10^4
  • s 只包含小写英文字母。

# 🧠 解题思路

由于删除的是子字符串，因此我们只需要分别考虑每一个连续的由 a 和 b 组成的子串。
思路：贪心
对于每一个由 a 和 b 组成的子串，使用括号匹配的方式，我们先选择得分更高的操作进行删除，再选择得分较低的操作进行删除，直到无法删除为止。
写法 1：栈
通过两个栈来进行两次遍历，第一次遍历时删除得分较高的操作，第二次遍历时删除得分较低的操作。
写法 2：计数
通过计数的方式，逐个统计每个子串中 a 和 b 的数量，进行匹配，匹配完高分的， a 和 b 的数量的最小值就是低分的操作可以删除的次数。
