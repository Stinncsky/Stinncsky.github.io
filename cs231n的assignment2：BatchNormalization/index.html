# 批归一化的数学原理

批归一化（Batch Normalization）是深度学习中用于加速训练和提高数值稳定性的关键技术。其核心思想是对每个特征在批次维度上进行标准化，消除内部协变量偏移。

# 前向传播的数学表达式

批归一化的前向传播包含四个核心步骤：

μ=1N∑i=1Nxi（样本均值）\mu = \frac{1}{N} \sum_{i=1}^{N} x_i \quad \text{（样本均值）}μ=N1​i=1∑N​xi​（样本均值）

σ2=1N∑i=1N(xi−μ)2（样本方差）\sigma^2 = \frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2 \quad \text{（样本方差）}σ2=N1​i=1∑N​(xi​−μ)2（样本方差）

x^=x−μσ2+ϵ（归一化）\hat{x} = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}} \quad \text{（归一化）}x^=σ2+ϵ​x−μ​（归一化）

y=γ⋅x^+β（缩放和偏移）y = \gamma \cdot \hat{x} + \beta \quad \text{（缩放和偏移）}y=γ⋅x^+β（缩放和偏移）

其中 NNN 为批大小，ϵ\epsilonϵ 为数值稳定项，γ\gammaγ 和 β\betaβ 为可学习参数。

批归一化将输入标准化为均值 0、方差 1 的分布，然后通过 γ\gammaγ 和 β\betaβ 恢复表达能力。

数值稳定性考虑

参数 ϵ\epsilonϵ（通常为 1e-5）防止分母为零，确保在方差极小时计算稳定。在实际实现中，这个小常数对梯度计算的影响通常可以忽略。

# 逐步梯度推导

反向传播遵循链式法则，从输出层向输入层逐步计算梯度。

# 步骤 1: ∂L∂β\frac{\partial L}{\partial \beta}∂β∂L​（最简单）

由于 y=γ⋅x^+βy = \gamma \cdot \hat{x} + \betay=γ⋅x^+β，有：

∂y∂β=1\frac{\partial y}{\partial \beta} = 1∂β∂y​=1

使用链式法则：

∂L∂β=∑i=1N∂L∂yi⋅∂yi∂β=∑i=1Ndouti\frac{\partial L}{\partial \beta} = \sum_{i=1}^{N} \frac{\partial L}{\partial y_i} \cdot \frac{\partial y_i}{\partial \beta} = \sum_{i=1}^{N} \text{dout}_i∂β∂L​=i=1∑N​∂yi​∂L​⋅∂β∂yi​​=i=1∑N​douti​

# 步骤 2: \frac{\partial L}

∂y∂γ=x^\frac{\partial y}{\partial \gamma} = \hat{x}∂γ∂y​=x^

∂L∂γ=∑i=1N∂L∂yi⋅∂yi∂γi=∑i=1Ndouti⋅x^i\frac{\partial L}{\partial \gamma} = \sum_{i=1}^{N} \frac{\partial L}{\partial y_i} \cdot \frac{\partial y_i}{\partial \gamma_i} = \sum_{i=1}^{N} \text{dout}_i \cdot \hat{x}_i∂γ∂L​=i=1∑N​∂yi​∂L​⋅∂γi​∂yi​​=i=1∑N​douti​⋅x^i​

# 步骤 3: ∂L∂x^\frac{\partial L}{\partial \hat{x}}∂x^∂L​（中间梯度）

∂y∂x^=γ\frac{\partial y}{\partial \hat{x}} = \gamma∂x^∂y​=γ

∂L∂x^=∂L∂y⋅∂y∂x^=dout⋅γ\frac{\partial L}{\partial \hat{x}} = \frac{\partial L}{\partial y} \cdot \frac{\partial y}{\partial \hat{x}} = \text{dout} \cdot \gamma∂x^∂L​=∂y∂L​⋅∂x^∂y​=dout⋅γ

# 步骤 4: ∂L∂σ2\frac{\partial L}{\partial \sigma^2}∂σ2∂L​（方差的梯度）

x^=x−μσ2+ϵ\hat{x} = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}}x^=σ2+ϵ​x−μ​

∂x^∂σ2=(x−μ)⋅(−12)⋅(σ2+ϵ)−3/2\frac{\partial \hat{x}}{\partial \sigma^2} = (x - \mu) \cdot \left(-\frac{1}{2}\right) \cdot (\sigma^2 + \epsilon)^{-3/2}∂σ2∂x^​=(x−μ)⋅(−21​)⋅(σ2+ϵ)−3/2

∂L∂σ2=∑i=1N∂L∂x^i⋅∂x^i∂σ2\frac{\partial L}{\partial \sigma^2} = \sum_{i=1}^{N} \frac{\partial L}{\partial \hat{x}_i} \cdot \frac{\partial \hat{x}_i}{\partial \sigma^2}∂σ2∂L​=i=1∑N​∂x^i​∂L​⋅∂σ2∂x^i​​

# 步骤 5: ∂L∂μ\frac{\partial L}{\partial \mu}∂μ∂L​（均值的梯度）

均值 μ\muμ 通过两个路径影响输出：

路径 1：直接通过 x^\hat{x}x^

∂x^∂μ=−1σ2+ϵ\frac{\partial \hat{x}}{\partial \mu} = \frac{-1}{\sqrt{\sigma^2 + \epsilon}}∂μ∂x^​=σ2+ϵ​−1​

路径 2：通过 σ2\sigma^2σ2

∂σ2∂μ=1N∑i=1N2(xi−μ)⋅(−1)=−2N∑i=1N(xi−μ)\frac{\partial \sigma^2}{\partial \mu} = \frac{1}{N} \sum_{i=1}^{N} 2(x_i - \mu) \cdot (-1) = \frac{-2}{N} \sum_{i=1}^{N} (x_i - \mu)∂μ∂σ2​=N1​i=1∑N​2(xi​−μ)⋅(−1)=N−2​i=1∑N​(xi​−μ)

# 步骤 6: ∂L∂x\frac{\partial L}{\partial x}∂x∂L​（最终梯度）

输入 xxx 通过三个路径影响输出：

路径 1：直接通过 x^\hat{x}x^

∂x^∂x=1σ2+ϵ\frac{\partial \hat{x}}{\partial x} = \frac{1}{\sqrt{\sigma^2 + \epsilon}}∂x∂x^​=σ2+ϵ​1​

路径 2：通过 σ2\sigma^2σ2

∂σ2∂x=2N(x−μ)\frac{\partial \sigma^2}{\partial x} = \frac{2}{N}(x - \mu)∂x∂σ2​=N2​(x−μ)

路径 3：通过 μ\muμ

∂μ∂x=1N\frac{\partial \mu}{\partial x} = \frac{1}{N}∂x∂μ​=N1​

# 链式法则的扩展

当一个变量通过多个路径影响最终输出时，总梯度等于各路径梯度的求和：

如果 zzz 通过多个中间变量 u1,u2,…,unu_1, u_2, \ldots, u_nu1​,u2​,…,un​ 影响 LLL，则：

∂L∂z=∂L∂u1⋅∂u1∂z+∂L∂u2⋅∂u2∂z+⋯+∂L∂un⋅∂un∂z\frac{\partial L}{\partial z} = \frac{\partial L}{\partial u_1} \cdot \frac{\partial u_1}{\partial z} + \frac{\partial L}{\partial u_2} \cdot \frac{\partial u_2}{\partial z} + \cdots + \frac{\partial L}{\partial u_n} \cdot \frac{\partial u_n}{\partial z}∂z∂L​=∂u1​∂L​⋅∂z∂u1​​+∂u2​∂L​⋅∂z∂u2​​+⋯+∂un​∂L​⋅∂z∂un​​

多路径原理的数学基础

这是偏导数线性性的直接应用。在计算图中，当一个节点有多个输出边时，其梯度是所有输出边梯度贡献的累加。这确保了梯度信息的完整传播。

# 标准版本的完整推导

标准的批归一化反向传播需要逐步计算各中间变量的梯度：

∂L∂β=∑i=1N∂L∂yi\frac{\partial L}{\partial \beta} = \sum_{i=1}^{N} \frac{\partial L}{\partial y_i}∂β∂L​=i=1∑N​∂yi​∂L​

∂L∂γ=∑i=1N∂L∂yi⋅x^i\frac{\partial L}{\partial \gamma} = \sum_{i=1}^{N} \frac{\partial L}{\partial y_i} \cdot \hat{x}_i∂γ∂L​=i=1∑N​∂yi​∂L​⋅x^i​

∂L∂x^=∂L∂y⋅γ\frac{\partial L}{\partial \hat{x}} = \frac{\partial L}{\partial y} \cdot \gamma∂x^∂L​=∂y∂L​⋅γ

这种方法需要 8 行代码实现。

# 优化版本的数学化简

通过数学推导，可以将复杂的多步计算合并为单一表达式：

∂L∂x=1Nσ2+ϵ[N⋅∂L∂x^−∑i=1N∂L∂x^i−x^∑i=1N∂L∂x^i⋅x^i]\frac{\partial L}{\partial x} = \frac{1}{N \sqrt{\sigma^2 + \epsilon}} \left[ N \cdot \frac{\partial L}{\partial \hat{x}} - \sum_{i=1}^{N} \frac{\partial L}{\partial \hat{x}_i} - \hat{x} \sum_{i=1}^{N} \frac{\partial L}{\partial \hat{x}_i} \cdot \hat{x}_i \right]∂x∂L​=Nσ2+ϵ​1​[N⋅∂x^∂L​−i=1∑N​∂x^i​∂L​−x^i=1∑N​∂x^i​∂L​⋅x^i​]

其中 ∂L∂x^=dout⋅γ\frac{\partial L}{\partial \hat{x}} = \text{dout} \cdot \gamma∂x^∂L​=dout⋅γ

优化公式的推导步骤

这个优化公式通过以下步骤得到：

  1. 展开所有路径的梯度表达式
  2. 利用 ∑i=1N(xi−μ)=0\sum_{i=1}^{N}(x_i - \mu) = 0∑i=1N​(xi​−μ)=0 的性质
  3. 合并同类项并提取公因子
  4. 将中间变量用归一化值 x^\hat{x}x^ 表示

最终将原本需要多步计算的过程压缩为单行代码。

# 批归一化的实际效果

# 对权重初始化的鲁棒性

在权重初始化规模变化时，批归一化展现出显著优势：

未使用 BN 的模型：

  • 当初始化规模过小（如 scale=0.01 ）时，模型收敛缓慢且性能差（梯度消失）
  • 当初始化规模过大（如 scale=0.1 ）时，训练不稳定，损失剧烈震荡（梯度爆炸）
  • 仅在极窄的初始化范围（如 scale=0.05 ）表现尚可

使用 BN 的模型：

  • 在极宽的初始化范围（如 scale=0.001 到 scale=1.0 ）内均能稳定训练
  • 损失快速收敛，且最终性能显著优于未使用 BN 的模型

批归一化通过标准化激活值，有效解决了深度网络中的梯度消失和梯度爆炸问题。

# 与批大小的关系

# 实验结果观察

小批大小场景（如 batch size=2/4/8）：

  • 无 BN 的模型：训练不稳定，收敛缓慢，最终准确率低
  • 使用 BN 的模型：
    • 当 batch size 极小时（如 2），性能显著下降甚至崩溃
    • 当 batch size≥8 时，仍能保持稳定训练

大批大小场景（如 batch size=256/512）：

  • 两种模型均稳定收敛
  • BN 模型的优势：
    • 收敛速度更快（快约 30%）
    • 最终准确率更高（约高 2-5%）

临界点现象：

  • BN 的增益在 batch size=16~128 区间最显著
  • 当 batch size<8 时，BN 可能损害性能
批大小与BN性能的关系机制

批归一化依赖批内统计量（均值和方差）进行标准化。当批大小过小时：

  1. 统计估计不准确：小样本的均值和方差波动大，无法准确代表数据分布
  2. 归一化噪声增加：不准确的统计量导致归一化过程引入噪声
  3. 训练不稳定：每个批次的标准化参数差异大，影响权重更新的一致性

因此，BN 需要足够大的批大小来获得稳定的统计估计。

# 层归一化 vs 批归一化

# 核心区别

  • 归一化维度：沿着批次维度计算统计量
  • 计算：对每个特征，在整个 batch 上计算均值和方差
  • 依赖性：依赖于 batch size，测试时需要运行时统计量
  • 适用场景: CNN、固定 batch size 的场景
  • 归一化维度：沿着特征维度计算统计量
  • 计算：对每个样本，在所有特征上计算均值和方差
  • 依赖性：不依赖 batch size，训练和测试行为一致
  • 适用场景: RNN、Transformer、变长序列
