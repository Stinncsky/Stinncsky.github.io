# 机器学习基础：从线性模型到神经网络

# 核心概念解析

# 超参数（Hyperparameters）

超参数是在训练过程开始之前设置的参数，它们控制学习过程本身，而不是通过训练数据学习得到的。与模型参数不同，超参数需要人工调整或通过超参数优化技术来确定。

超参数详细解释

常见的超参数包括：

  • 学习率（Learning Rate）：控制参数更新的步长
  • 批次大小（Batch Size）：每次梯度更新使用的样本数量
  • 隐藏层数量和神经元数量：决定网络的架构复杂度
  • 正则化参数：防止过拟合的强度控制
  • 优化器选择：如 SGD、Adam 等算法的选择

超参数的选择直接影响模型的性能和训练效率，通常需要通过网格搜索、随机搜索或贝叶斯优化等方法来寻找最优组合。

# 模型偏差（Model Bias）

模型偏差是指模型在所有可能的训练集上的期望预测与真实值之间的差异。它反映了模型的假设与真实数据生成过程之间的差距。

偏差的影响和权衡
  • 高偏差：模型过于简单，无法捕捉数据中的复杂模式，导致欠拟合
  • 低偏差：模型复杂度适中或较高，能够较好地拟合真实的数据分布
  • 偏差 - 方差权衡：在机器学习中，我们需要在偏差和方差之间找到平衡点
  • 减少偏差的方法：增加模型复杂度、添加更多特征、减少正则化强度

# 批次（Batch）

批次是指在一次前向传播和反向传播过程中，同时处理的训练样本数量。批次大小是一个重要的超参数，影响训练速度和模型性能。

不同批次大小的特点
  • 批次梯度下降（Batch GD）：使用全部训练数据，梯度估计准确但计算慢
  • 随机梯度下降（SGD）：批次大小为 1，计算快但梯度估计噪声大
  • 小批次梯度下降（Mini-batch GD）：折中方案，通常批次大小为 32、64、128 等
  • 大批次的优势：更稳定的梯度估计，更好地利用并行计算
  • 小批次的优势：更频繁的参数更新，有助于跳出局部最优

# 轮（Epoch）

一个轮（epoch）是指整个训练数据集被模型完整地训练一遍的过程。在一个 epoch 中，所有训练样本都会被用于模型的参数更新。

轮与批次的关系
  • 一轮包含多个批次：当数据集较大时，一个 epoch 会被分解为多个 batch 进行处理
  • 参数更新频率：每个 batch 都会进行一次参数更新，因此一个 epoch 内会有多次参数更新
  • 计算示例：假设训练集有 1000 个样本，批次大小为 100，则一个 epoch 包含 10 个 batch
  • 内存管理：通过批次处理可以在有限内存下训练大规模数据集
训练轮数的意义
  • 多轮训练的必要性：复杂模型通常需要多次遍历数据才能学到有效的表示
  • 早停法（Early Stopping）：通过监控验证集性能来决定何时停止训练
  • 学习曲线：通过观察训练和验证损失随 epoch 的变化来诊断模型状态
  • 过拟合监控：当验证损失开始上升时，可能出现了过拟合

# 深度学习（Deep Learning）

深度学习是机器学习的一个分支，使用具有多个隐藏层的神经网络来学习数据的复杂表示。"深度" 指的是网络中隐藏层的数量。

深度学习的核心优势
  • 自动特征学习：无需手工设计特征，网络自动学习数据的有效表示
  • 层次化表示：浅层学习简单特征，深层学习复杂的抽象概念
  • 强大的表达能力：理论上可以逼近任意复杂的函数
  • 广泛应用：在图像识别、自然语言处理、语音识别等领域取得突破性进展
  • 端到端学习：从原始数据直接学习到最终输出的映射关系

# 机器学习基础任务

机器学习中存在两种基本任务类型：

  • 回归任务：预测连续数值输出，如房价预测、股票价格预测
  • 分类任务：预测离散类别标签，如图像识别、垃圾邮件检测

# 损失函数

损失函数用于衡量模型预测值与真实值之间的差距：

# 回归任务常用损失函数

  • 均方误差（MSE）：适用于输出值变化范围较大的回归问题
  • 平均绝对误差（MAE）：对异常值更加鲁棒

# 分类任务常用损失函数

  • 交叉熵损失（Cross-Entropy Loss）：最常用的分类损失函数

# 优化算法

梯度下降是用于最小化损失函数的优化算法：

  • 学习率控制参数更新步长，需根据模型复杂度和数据集大小进行调整
  • 学习率过大可能导致训练不稳定，过小则训练速度慢

# 机器学习一般步骤

# 激活函数

激活函数是神经网络中用于引入非线性特性的数学函数：

# Sigmoid 函数

  • 特点：S 型曲线激活函数，将任意实数映射到 (0,1) 区间内
  • 优势：输出范围有界，便于概率解释
  • 劣势：存在梯度消失问题

# ReLU 函数

  • 特点：分段线性函数，定义为 max (0,x)
  • 优势：计算简单，有效缓解梯度消失问题
  • 劣势：可能出现神经元死亡现象
