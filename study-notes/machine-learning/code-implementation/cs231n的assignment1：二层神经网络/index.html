# 前向传播与反向传播基础

# 前向传播与反向传播的核心概念

前向传播和反向传播是神经网络训练的两个核心过程，它们构成了梯度下降优化的完整循环。

# 前向传播（Forward Propagation）

  • 目的：根据当前参数计算模型的预测输出
  • 方向：从输入层到输出层
  • 输出：损失值和中间激活值

前向传播过程中，输入数据按层依次传递，每层执行线性变换和激活函数操作，最终得到预测结果。

# 反向传播（Backward Propagation）

  • 目的：计算损失函数对各参数的梯度
  • 方向：从输出层到输入层
  • 输出：各层参数的梯度

反向传播利用链式法则，从输出层开始逐层计算梯度，为参数更新提供方向信息。

前向传播与反向传播的数学基础

前向传播实质是复合函数的计算过程：y=fn(fn−1(...f1(x)))y = f_n(f_{n-1}(...f_1(x)))y=fn​(fn−1​(...f1​(x)))，其中每个 fif_ifi​ 代表一层的变换。

反向传播则是复合函数求导的链式法则应用：

∂L∂θi=∂L∂y⋅∂y∂fn⋅...⋅∂fi∂θi\frac{\partial L}{\partial \theta_i} = \frac{\partial L}{\partial y} \cdot \frac{\partial y}{\partial f_n} \cdot ... \cdot \frac{\partial f_i}{\partial \theta_i}∂θi​∂L​=∂y∂L​⋅∂fn​∂y​⋅...⋅∂θi​∂fi​​

这种设计使得深度网络能够高效地计算高维参数空间中的梯度信息。

# 仿射变换的数学原理与实现

# 仿射变换的数学表示

仿射变换是神经网络中最基础的线性变换操作，其数学形式为：

out=X⋅W+b\text{out} = X \cdot W + bout=X⋅W+b

其中：

  • X∈RN×DX \in \mathbb{R}^{N \times D}X∈RN×D：输入矩阵（N 个样本，D 个特征）
  • W∈RD×MW \in \mathbb{R}^{D \times M}W∈RD×M：权重矩阵
  • b∈RMb \in \mathbb{R}^{M}b∈RM：偏置向量
  • out∈RN×M\text{out} \in \mathbb{R}^{N \times M}out∈RN×M：输出矩阵
仿射变换的几何意义

仿射变换结合了线性变换和平移操作：

  • 线性变换部分 X⋅WX \cdot WX⋅W：实现空间的旋转、缩放和剪切
  • 平移部分 +b+b+b：将变换后的结果在各维度上进行偏移

在神经网络中，仿射变换负责将输入特征映射到新的特征空间，为后续的非线性激活函数提供输入。每一层的仿射变换都在学习如何最优地组合输入特征。

# 梯度计算原理

假设上游梯度是 dout （形状 (N, M) ），需要计算损失函数对各参数的梯度：

  • dx ：损失对输入 x 的梯度
  • dw ：损失对权重 w 的梯度
  • db ：损失对偏置 b 的梯度

由于偏置 bbb 被广播到每个样本，所以 dbdbdb 是所有样本梯度的累加：

# 对权重 w 的梯度推导

∂L∂wk,j=∑i=1N∂L∂outi,j⋅∂outi,j∂wk,j=∑i=1Ndouti,j⋅xi,k\frac{\partial L}{\partial w_{k,j}} = \sum_{i=1}^{N} \frac{\partial L}{\partial \text{out}_{i,j}} \cdot \frac{\partial \text{out}_{i,j}}{\partial w_{k,j}} = \sum_{i=1}^{N} \text{dout}_{i,j} \cdot x_{i,k}∂wk,j​∂L​=i=1∑N​∂outi,j​∂L​⋅∂wk,j​∂outi,j​​=i=1∑N​douti,j​⋅xi,k​

# ReLU 激活函数实现

ReLU（Rectified Linear Unit）是目前最常用的激活函数，其数学定义为：

ReLU(x)=max⁡(0,x)\text{ReLU}(x) = \max(0, x)ReLU(x)=max(0,x)

# 激活函数的梯度消失分析

不同激活函数在梯度消失问题上表现不同：

激活函数梯度消失情况原因分析
Sigmoid✅ 存在函数两端饱和区梯度趋近于 0
ReLU✅ 存在负区间梯度恒为 0，导致神经元 "死亡"
Leaky ReLU❌ 不存在负区间引入小斜率（如 0.01）避免零梯度
Leaky ReLU的设计优势

Leaky ReLU 的数学定义为：

Leaky ReLU(x)={xifx>0αxifx≤0\text{Leaky ReLU}(x) = \begin{cases} x & \text{if } x > 0 \\ \alpha x & \text{if } x \leq 0 \end{cases}Leaky ReLU(x)={xαx​if x>0if x≤0​

其中 α\alphaα 是一个小的正值（通常为 0.01）。

相应的梯度函数为：

Leaky ReLU′(x)={1ifx>0αifx≤0\text{Leaky ReLU}'(x) = \begin{cases} 1 & \text{if } x > 0 \\ \alpha & \text{if } x \leq 0 \end{cases}Leaky ReLU′(x)={1α​if x>0if x≤0​

通过在负区间保留 α\alphaα 梯度，Leaky ReLU 从根本上解决了 ReLU 的 "死亡神经元" 问题，使得负输入的神经元仍能参与梯度传播和参数更新。

关键结论：Sigmoid 的饱和性和 ReLU 的负值硬截断是梯度消失的主因；Leaky ReLU 通过负区间引入微小线性激活解决了此缺陷。

# Softmax 损失函数的数学原理

Softmax 损失函数结合了 softmax 激活和交叉熵损失，常用于多分类问题。对于输入分数 x∈RN×Cx \in \mathbb{R}^{N \times C}x∈RN×C，softmax 概率计算为：

pi,j=exp⁡(xi,j−max⁡kxi,k)∑k=1Cexp⁡(xi,k−max⁡kxi,k)p_{i,j} = \frac{\exp(x_{i,j} - \max_k x_{i,k})}{\sum_{k=1}^{C} \exp(x_{i,k} - \max_k x_{i,k})}pi,j​=∑k=1C​exp(xi,k​−maxk​xi,k​)exp(xi,j​−maxk​xi,k​)​

交叉熵损失为：

L=−1N∑i=1Nlog⁡(pi,yi)L = -\frac{1}{N} \sum_{i=1}^{N} \log(p_{i,y_i})L=−N1​i=1∑N​log(pi,yi​​)

在 softmax 计算中，减去最大值 np.max(x, axis=1, keepdims=True) 是为了数值稳定性，防止指数运算导致的溢出问题。

# 过拟合问题与解决方案

当测试准确率远低于训练准确率时，表明模型出现过拟合。以下是减小训练 - 测试差距的有效方法：

方法有效性原理分析
使用更大的数据集训练✅扩大数据集能提供更多样化的样本，降低模型对训练集特定噪声的过拟合，提升泛化能力
增加隐藏单元数量❌增加网络复杂度（更多参数）会加剧过拟合，使训练准确率更高但测试差距更大
增强正则化强度✅更强的正则化（如增大 L2 权重衰减 / Dropout 比例）能抑制过拟合，强制模型学习更鲁棒的特征

实践建议：优先考虑数据增强和正则化技术，避免盲目增加模型复杂度。
